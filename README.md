EmailSenderAPI(одногруппника) - обеспечивает надежную отправку электронных писем с HTML-контентом через SMTP-сервер.

WeatherForecastAPI(моё) - предоставляет актуальные прогнозы погоды для различных городов в структурированном JSON-формате.

Логика работы веб-сервиса - пользователь открывает интуитивно понятный веб-интерфейс, где видит простую форму с полем для email и выбором города. После ввода своего электронного адреса и выбора интересующего города система автоматически запрашивает актуальный прогноз погоды, красиво оформляет его в виде информативного письма и моментально отправляет прямо в почтовый ящик пользователя. Весь процесс происходит в фоновом режиме — пользователь просто получает готовый прогноз в удобном формате без необходимости переходить между разными сервисами или вручную обрабатывать данные.
